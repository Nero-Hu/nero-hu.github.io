Trace the smallest element in a specially designed stack

Apr 10, 2014

Question:

Design a stack, which owns a min() function that returns the smallest elements within the stack.

Additional requirements

Time complexity of push(), pop() and min() function should be O(1).

Solution

Code

The point should be in the function of push() shown as following code.

public NodeWithMin push(NodeWithMin item) {
    // TODO Auto-generated method stub
    if (this.isEmpty()){
        item.MinNode = item.Node;
        return super.push(item);
    }
    else{
        if ( item.Node > this.peek().MinNode ) {
            item.MinNode = this.peek().MinNode;
            return super.push(item);
        }
        else{
            item.MinNode = item.Node;
            return super.push(item);
        }
    }
}
