Deploy Rime[Squirrel] on OS X

Apr 18, 2014

什么是中州韻

中州韻輸入法引擎，思想用鍵盤表達也行。

如何安装

下载Squirrel(Rime Mac Version)并且安装.pkg文件。

基础设置

自定义输入配置文件

cd ~/Library/Rime
touch squirrel.custom.yaml

编辑squirrel.custom.yaml.

patch:
  style:
    horizontal: true
    font_point: 14

此配置将会覆盖默认安装目录下的default.yaml文件内的相应设置。

比如这里就将候选字体的大小设置为14，并且将候选模式设置为水平。

用户数据同步

cd ~/Library/Rime
vi installation.yaml

sync_dir: '/Dropbox/Rime-Settings'

** 修改上述设置后，需要重新部署squirrel才会生效。
