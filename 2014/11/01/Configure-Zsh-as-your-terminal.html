Configure zsh as your terminal on MAC

Nov 1, 2014

After years of using Bash, I finally switch my default terminal to Zsh, which give me a way better experience, including more powerful autocomplete and directory jump.

Actually there are a lot of existing solutions such as oh-my-zsh, but it is too heavy. I don't want to many colorful prompts or autocorrections. I just need the pure simple useful functions that I want, so I customized my own profile with only one plugin autojump. So far so good.

Use Zsh as your default terminal

chsh -s zsh

Set up your own zsh

vi ~/.zshrc

Here are my settings.

#alias settings below
alias ls='ls -G'
alias ll='ls -lah'

#Autojump
[[ -s /Users/nerohu/.autojump/etc/profile.d/autojump.sh ]] && source /Users/nerohu/.autojump/etc/profile.d/autojump.sh

#Zsh settings
CASE_SENSITIVE="true"
ENABLE_CORRECTION="true"

#PROMPT settings:
PROMPT="%n@%m:%c$ "

#Command completion
autoload -U compinit
compinit
".zshrc" 42L, 1132C
#Command completion
autoload -U compinit
compinit
zstyle ':completion:*:descriptions' format '%U%B%d%b%u'
#kill completion
#zstyle ':completion:*:*:kill:*:processes' list-colors '=(#b) #([0-9]#)*=0=01;31'
#zstyle ':completion:*:kill:*' command 'ps -u $USER -o pid,%cpu,tty,cputime,cmd'
zstyle ':completion:*' list-prompt %SAt %p: Hit TAB for more, or the character to insert%s
#History size
export HISTSIZE=10000
export SAVEHIST=10000
#History file location
export HISTFILE=~/.zshistory
#Append
setopt INC_APPEND_HISTORY
#Ignore continous duplicate
setopt HIST_IGNORE_DUPS

#Bind keys
bindkey '^[[H' beginning-of-line
bindkey '^[[F' end-of-line
bindkey "^[[3~" delete-char
